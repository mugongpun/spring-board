# 게시판 웹 애플리케이션

이 프로젝트는 게시판 웹 애플리케이션입니다. 사용자는 회원가입, 로그인, 게시글 작성 및 목록 조회 기능을 이용할 수 있습니다.

## 주요 화면

### 1. 게시판 메인 화면
게시판의 전체적인 구조를 확인할 수 있는 화면입니다.
